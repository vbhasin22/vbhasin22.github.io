Final Project: Securing the Internet

My final project is about secure data transmission over the internet and how it can be improved in the connected age:

  • HTTPS and SSL/TLS: The current method
    • Encrypt all your communication with a server who's identity has been verified by a third party certificate authority
  • Issues with HTTPS
    • Trust Model relies a lot on the Third Party Certificate authority which can be influenced by external sources.
    • Susceptible to MITM attacks
  • Emerging or Theoretical ways to make the internet more secure
    • Using Bitcoin's and others Blockchain
